Debian Base Package Installer

Description

This script installs your packages and programs on debian bases linux distro.

Usage

Step 1:

Download zip file from github

or clone from github:

git clone https://github.com/nimaebrazi/debian-base-packeage-installer.git
Step 2:

Add your package(s) to config/programs.list. Every package must be in one line. Sample: 

apache2 
apache2-doc
mysql-server 
libapache2-mod-auth-mysql
php5
.
.
.
Step 3:

change directory to src:

cd src/

Note: All .sh files must be executable:

sudo chmod +x *.sh
Step 4:

Just run main.sh in src directory

bash main.sh

or

./main.sh
